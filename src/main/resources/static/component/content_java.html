第 1 章：Java 简介

Java 是一种面向对象的编程语言，由Sun Microsystems的James Gosling于1995年发布。Java 语言具有平台无关性和安全性。

第 2 章：Java 基础语法

Java 的基本语法与 C++ 类似，程序由类和方法组成，最基本的方法是 main() 方法。

public class Main {
    public static void main(String[] args) {
        // 程序入口
        System.out.println("Hello, World!");
    }
}

第 3 章：数据类型

Java 支持多种数据类型，包括基本数据类型和引用数据类型。

  • 基本数据类型：int, char, float, double, boolean
  • 引用数据类型：类, 接口, 数组

第 4 章：控制结构

控制结构用于控制程序的执行流程，包括条件语句和循环语句。

条件语句

if (condition) {
    // 执行代码
} else {
    // 执行其他代码
}

循环语句

for (int i = 0; i < 10; i++) {
    // 执行代码
}

第 5 章：方法

方法是执行特定任务的代码块，可以提高代码的重用性。

public int add(int a, int b) {
    return a + b;
}

第 6 章：类与对象

Java 是一种面向对象的编程语言，支持类和对象的概念。

public class Dog {
    public void bark() {
        System.out.println("Woof!");
    }

    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.bark();
    }
}

第 7 章：异常处理

Java 提供了异常处理机制来处理运行时错误。

try {
    // 可能抛出异常的代码
} catch (Exception e) {
    // 处理异常
    e.printStackTrace();
}

第 8 章：集合框架

Java 集合框架提供了许多常用的数据结构和算法。

import java.util.ArrayList;

ArrayList numbers = new ArrayList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

第 9 章：文件操作

Java 提供了文件输入输出的功能，可以方便地读写文件。

import java.io.FileWriter;
import java.io.IOException;

public class FileOperations {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("output.txt");
            writer.write("Hello, World!");
            writer.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
